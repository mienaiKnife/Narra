# Project Roadmap

## MVP Scope (Current Target)
- RSS feed subscription and article fetching.
- EPUB file importing and parsing.
- Web page importing with reader-mode extraction.
- Listening queue/playlist UI.
- TTS playback using Android's built-in engine.
- On-device AI TTS using Sherpa-ONNX.
- OPML export/import.
- File-based backup and restore.
- Foreground service for background playback with media controls.

## Planned Features (Future)
- PDF file importing and parsing.
- Support for self-hosted AI TTS servers (Kokoro, Coqui, Piper).
- Additional cloud AI TTS providers.
- Multi-platform support (Desktop, iOS via KMP).
- Optional sync via self-hosted servers (Nextcloud/gpodder).
- Improvements to readability/reader-mode heuristics.
- Import text via photo scanning.
- Customizable color themes.
